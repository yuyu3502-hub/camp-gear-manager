# お父さんのキャンプ用品管理アプリ

お父さんが、トランクルーム内のキャンプ用品を「どこに置いたか」と「どのキャンプに持っていくか」で管理するWebアプリです。
棚の段、右奥、床手前などの保管場所と、ケース・バッグ単位の用品を写真付きで記録できます。

公開ページ:
https://yuyu3502-hub.github.io/camp-gear-manager/

## はじめて使う流れ

1. **トランクルーム内の場所を登録する**
   - 最初は保管場所・用品・セットが0件の状態から始まります。
   - 例: 左ラック上段、左ラック2段目、右奥、床手前
   - 現地で見てすぐ分かる名前にしておくと、出発前に探しやすくなります。

2. **用品を追加する**
   - 用品名、カテゴリ、数量、状態、メモ、写真を登録します。
   - 最初は「黒いハードケース」「緑のバッグ」など、ケース・バッグ単位で登録しても問題ありません。
   - 中身が分からないものは、状態を「中身未確認」にして仮登録します。
   - 中身が分かったものだけ、後から正確な名前に直します。

3. **キャンプ別の持ち出しセットを作る**
   - 例: ファミリー1泊、デイキャンプ、ソロ軽量
   - 持っていく用品にチェックを入れて保存します。

4. **出発前にチェックする**
   - トップ画面で持ち出しセットを選びます。
   - 持ち物チェックを付けながら、どの保管場所へ取りに行けばよいか確認します。

5. **バックアップファイルを保存する**
   - 「データ管理」を開き、「バックアップを保存」からバックアップファイルを保存できます。
   - 別の端末で使うときは「バックアップから復元」で読み込んでください。

## 主な機能

- 保管場所の追加・編集・削除
- 用品の追加・編集・削除
- 用品写真の登録
- キャンプ別の持ち出しセット管理
- 持ち物チェックリスト
- 検索
- バックアップファイルの保存/復元
- スマホ対応レイアウト

## データ保存について

このアプリのデータは、開いているブラウザの中に保存されます。

- 用品、保管場所、持ち出しセット: `localStorage`
- 写真: `IndexedDB`

同じ端末・同じブラウザ・同じURLで開き直す場合は保存されています。
ただし、別端末や別ブラウザには自動共有されません。

消える可能性がある操作:

- ブラウザの履歴やサイトデータを削除する
- プライベートブラウズで使う
- 別のURLで開く
- 別の端末や別ブラウザで開く

大事なデータを入れたら、定期的に「バックアップを保存」でバックアップファイルを保存してください。

## ローカルで開く

`index.html` を直接ブラウザで開くか、ローカルサーバーで配信します。

```bash
python3 -m http.server 8788 --bind 0.0.0.0
```

同じWi-Fi内の端末から開く場合は、MacのLAN IPを使います。

```text
http://<MacのLAN IP>:8788/
```
