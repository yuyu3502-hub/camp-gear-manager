# キャンプ用品収納箱管理アプリ

キャンプ用品を「どの収納箱（コンテナ）に入っているか」と「どのキャンプに持っていくか」で管理するWebアプリです。
ファミリーキャンプ、ソロキャンプ、デイキャンプなど、行き先ごとの持ち物セットを作れます。

公開ページ:
https://yuyu3502-hub.github.io/camp-gear-manager/

## はじめて使う流れ

1. **収納箱（コンテナ）を登録する**
   - 例: キッチン箱、照明・電源箱、設営道具箱
   - 置き場所も入れておくと、出発前に探しやすくなります。

2. **アイテムを追加する**
   - アイテム名、カテゴリ、数量、状態、メモ、写真を登録します。
   - 写真を入れておくと、家族でも同じものを見つけやすくなります。

3. **キャンプ別の持ち出しセットを作る**
   - 例: ファミリー1泊、デイキャンプ、ソロ軽量
   - 持っていくアイテムにチェックを入れて保存します。

4. **出発前にチェックする**
   - トップ画面で持ち出しセットを選びます。
   - 持ち物チェックを付けながら、積み込み先の収納箱を確認します。

5. **バックアップファイルを保存する**
   - 右上の「バックアップを保存」からバックアップファイルを保存できます。
   - 別の端末で使うときは「バックアップから復元」で読み込んでください。

## 主な機能

- 収納箱（コンテナ）の追加・編集・削除
- アイテムの追加・編集・削除
- アイテム写真の登録
- キャンプ別の持ち出しセット管理
- 持ち物チェックリスト
- 検索
- バックアップファイルの保存/復元
- スマホ対応レイアウト

## データ保存について

このアプリのデータは、開いているブラウザの中に保存されます。

- アイテム、収納箱、持ち出しセット: `localStorage`
- 写真: `IndexedDB`

同じ端末・同じブラウザ・同じURLで開き直す場合は保存されています。
ただし、別端末や別ブラウザには自動共有されません。

消える可能性がある操作:

- ブラウザの履歴やサイトデータを削除する
- プライベートブラウズで使う
- 別のURLで開く
- 別の端末や別ブラウザで開く

大事なデータを入れたら、定期的に「バックアップを保存」でバックアップファイルを保存してください。

## ローカルで開く

`index.html` を直接ブラウザで開くか、ローカルサーバーで配信します。

```bash
python3 -m http.server 8788 --bind 0.0.0.0
```

同じWi-Fi内の端末から開く場合は、MacのLAN IPを使います。

```text
http://<MacのLAN IP>:8788/
```
