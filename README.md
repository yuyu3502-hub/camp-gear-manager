# キャンプ用品コンテナ管理アプリ

キャンプ用品をコンテナ別に管理し、キャンプの種類ごとの持ち出しセットを作れる静的Webアプリです。

## 機能

- コンテナの追加・編集・削除
- アイテムの追加・編集・削除
- アイテム写真の登録
- キャンプ別プリセット管理
- 持ち物チェックリスト
- JSONバックアップ/復元
- スマホ対応レイアウト

## 使い方

`index.html` をブラウザで開くか、ローカルサーバーで配信します。

```bash
python3 -m http.server 8788 --bind 0.0.0.0
```

同じWi-Fi内の端末からは、MacのLAN IPを使って開けます。

## データ保存

アイテムやプリセットなどのメタデータはブラウザの `localStorage`、写真は `IndexedDB` に保存します。
端末やブラウザごとに保存されるため、別端末と共有したい場合はバックアップ/復元を使ってください。

## GitHub Pages

静的ファイルだけで動くため、GitHub Pagesにそのまま公開できます。

1. GitHubで新しいリポジトリを作成
2. このフォルダをpush
3. Repository Settings > Pages
4. Sourceを `Deploy from a branch`、Branchを `main` / `/root` に設定

公開後もデータは各ブラウザ内に保存されます。
